# CumaPro Privacy Policy Site

Static privacy policy page for CumaPro by **Mezsondra LTD**.

## Included files

- `index.html` – the privacy policy page, including disclosures for no app data collection and AdMob ads.
- `logo.svg` – logo shown in the page header.

## Deploy to Vercel

1. Push this repository to GitHub.
2. Import the repository in Vercel.
3. Deploy with default settings.

Vercel will serve `index.html` as the main page.
